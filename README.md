# Panqueco Café Landing Page

Landing page premium para cafeteria, desenvolvida com foco em identidade visual forte, experiência visual refinada e navegação fluida.

## ✨ Visão geral

Este projeto foi criado para apresentar uma cafeteria de forma moderna e envolvente, destacando:
- proposta da marca
- cardápio
- ambiente
- experiência do cliente
- apelo visual sofisticado

A interface foi construída para transmitir um ar artesanal, acolhedor e premium, com atenção especial aos detalhes visuais e à responsividade.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Principais características

- layout moderno e responsivo
- identidade visual elegante
- seções bem distribuídas
- animações e microinterações
- cursor personalizado
- selo circular decorativo em SVG
- navegação fluida
- estrutura leve e direta
